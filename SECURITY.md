# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately rather than opening a public GitHub issue.

**Email:** aayushirajesh7@gmail.com

Include:

- Description of the vulnerability
- Steps to reproduce
- Potential impact

I'll acknowledge reports within a few days and work on a fix as soon as possible.

---

## Scope

This applies to:

- Authentication (Supabase Auth)
- Row Level Security and data access
- Handling of user data on the client
